# Problem Statement

Build a beginner-friendly **REST API learning project** using **Node.js** that provides practical experience with:

1. Designing RESTful endpoints for a `Product` resource.
2. Implementing full CRUD operations (`GET`, `POST`, `PUT`, `PATCH`, `DELETE`).
3. Using a **local database** (SQLite) to store and retrieve data.
4. Connecting API endpoints to a website built with **HTML + CSS + JavaScript**.
5. Testing endpoints with **Postman** collection and examples.
6. Documenting setup, usage, and best practices clearly for learners.

## Functional Requirements

- Expose `/api/products` endpoints.
- Validate request payloads and return proper HTTP status codes.
- Support query filtering, search, sorting, and pagination.
- Serve a frontend from the same Node.js app.
- Provide seed data for quick testing.

## Non-Functional Requirements

- Keep project structure simple and beginner-friendly.
- Use predictable response JSON format.
- Include clear error handling.
- Include complete README and hands-on exercises.

## Expected Outcome

A learner should be able to:

- run the project locally,
- create/read/update/delete products,
- inspect responses in browser and Postman,
- understand how frontend communicates with backend API and local database.
